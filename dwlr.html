Projet DWLR

Description

Système de mesure de niveau d’eau avec transmission GSM.

Composants

  • ESP32
  • SIM800L
  • ADS1115
  • Carte SD

Fonctionnement

Lecture analogique → conversion ADC → envoi via GSM → stockage SD

Défis techniques

Gestion UART avec SIM800L, stabilité alimentation, gestion erreurs AT.
